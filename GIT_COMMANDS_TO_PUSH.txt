================================================================================
IF YOU GET AUTHENTICATION ERROR
================================================================================

Try this command to use Personal Access Token:

git remote remove origin

git remote add origin https://TOKEN@github.com/YOUR_USERNAME/agent_test.git

git push -u origin main

Where TOKEN is your GitHub Personal Access Token from:
https://github.com/settings/tokens

================================================================================
TROUBLESHOOTING
================================================================================

"fatal: remote origin already exists"
→ Run: git remote remove origin
→ Then add new remote with correct URL

"fatal: 'origin' does not appear to be a 'git' repository"
→ Wrong URL format
→ Use: https://github.com/USERNAME/agent_test.git

"Permission denied (publickey)"
→ SSH key issue
→ Use HTTPS URL instead of SSH
→ Or configure SSH keys in GitHub settings

"remote: Repository not found"
→ Wrong repository URL
→ Verify repository name matches exactly
→ Check username spelling
