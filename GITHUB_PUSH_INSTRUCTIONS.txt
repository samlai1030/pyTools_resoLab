TROUBLESHOOTING
================================================================================

If you get "authentication failed":
- Verify you entered correct GitHub username/password or token
- Check if 2FA is enabled (you'll need Personal Access Token)
- Try: git remote set-url origin https://TOKEN@github.com/USERNAME/agent_test.git

If you get "remote already exists":
- Run: git remote remove origin
- Then add new remote with correct URL

If push is rejected:
- Run: git pull origin main --allow-unrelated-histories
- Then: git push -u origin main
